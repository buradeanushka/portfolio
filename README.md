# Anushka Burade Portfolio

## Overview
This is my personal portfolio website built using **Next.js**, **NUI**, and **NUICSS**. It showcases my skills, projects, education, and contact information in a clean and responsive design.

## Features
- Responsive Design
- Hero Section
- About Me
- Skills & Tech Stack
- Projects Section
- Contact Information
- Resume Download

## Tech Stack
- Next.js
- TypeScript
- NUI
- NUICSS
- CSS
